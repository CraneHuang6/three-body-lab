# 三体实验室

一个以《三体》为灵感的 Mac 桌面互动作品。

你可以直接观看八大天象演示，也可以自己调 4 个天体的参数，看轨迹和碰撞怎么变化。

## 下载

请前往发布页下载最新版本：

[下载最新版本](https://github.com/CraneHuang6/three-body-lab/releases/latest)

普通用户请优先下载：

- `ThreeBodyLab-0.1.0-mac-arm64.zip`

## 适用系统

当前版本适用于：

- macOS
- Apple Silicon 芯片（M1 / M2 / M3 / M4）

## 安装步骤

1. 下载 `zip` 文件
2. 双击解压
3. 把 `ThreeBodyLab.app` 拖进“应用程序”
4. 打开“应用程序”里的 `ThreeBodyLab`

## 第一次打不开怎么办

因为这是免费分发版本，第一次打开时 macOS 可能会拦截。这是正常现象，不代表文件坏了。

按下面做即可：

1. 在“应用程序”里找到 `ThreeBodyLab`
2. 按住 `control` 键点它
3. 选择“打开”
4. 再点一次“打开”

通常第一次这样处理后，后面就能正常启动。

如果系统仍然拦截，也可以去：

1. “系统设置”
2. “隐私与安全性”
3. 在底部找到 `ThreeBodyLab` 的提示
4. 点击“仍要打开”

## 打开后怎么玩

应用首页有两个入口：

- `观演模式`：直接观看八大天象演示
- `实验室模式`：手动调整 4 个天体参数，观察新的轨迹和碰撞效果

第一次体验建议顺序：

1. 先看 `观演模式`
2. 再进入 `实验室模式`

这样更容易理解这个作品原本的演示效果，以及你可以自己动手修改的部分。
